About Me

My name is Mike Simpson (aka MDSimpson or Codemonkey_Mike), and I'm a Research Software Engineer and former Game Developer, based in the North-East of England.

  • Trustee and Vice-President of the Society of Research Software Engineering (2023-present).
  • A 2025 Fellow of the Software Sustainability Institute.

About this site

This site is a personal website and blog, where I share my thoughts on software engineering, research software, and game development. It is built using Jekyll and hosted on GitHub Pages.

The site has been exported from WordPress, and is still a work in progress. I am still working on migrating content from the old site, and updating the design and layout of the site.

The site was developed using Jekyll, a static site generator, and is hosted on GitHub Pages. The source code for the site is available on GitHub.
